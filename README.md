# sheikh-yusuf-lokoja-bibliographic-database

## Bibliographic Database of the Works of Sheikh Yusuf Lokoja

This repository contains a structured bibliographic dataset of all known works of Sheikh Yusuf Lokoja, developed as part of academic research on the digital cataloguing and preservation of Islamic scholarly writings in Nigeria.

The records were compiled from documentary sources and organized using a relational database approach. The original data were transformed from unstructured documents into structured tables through SQL queries, then exported into Excel and saved in CSV format to ensure accessibility, reuse, and long term preservation.

The dataset captures key bibliographic information including title of work, author attribution, subject classification, language, publication details, and source references. It serves as a digital catalog dedicated specifically to the scholarly contributions of Sheikh Yusuf Lokoja.

-----

## Purpose of the Dataset

The purpose of this dataset is to provide researchers and scholars with an organized, searchable, and reusable bibliographic resource focused on the works of Sheikh Yusuf Lokoja. It demonstrates how database systems can be applied to preserve and disseminate Islamic intellectual heritage.

----

## Data Source

The records were obtained from verified catalogs, manuscripts, and documentary materials relating to the works of Sheikh Yusuf Lokoja for academic research.

----

## File Format

The data is provided in CSV format for easy import into database systems, spreadsheet software, and analytical tools.

----

## How to Use

Researchers can download the CSV file and import it into SQL, Excel, Python, R, or any database management system for querying, analysis, or further catalog development.
